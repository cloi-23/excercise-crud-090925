# Simple CRUD App

A **full-stack CRUD application** built with **NestJS** backend, **MongoDB**, and **React** frontend. Employees can be added, edited, deleted, and photos are stored using **MongoDB GridFS**.

## Setup

Clone the repository:

```bash
cd simple-crud
```

# Backend
```bash
cd backend-server
npm install
npm run start:dev
```
# Frontend
```bash
cd ../web-client
npm install
npm run start
```
